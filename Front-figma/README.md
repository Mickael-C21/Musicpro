# Sans titre

This is a code bundle for Sans titre. The original project is available at https://www.figma.com/design/S7p1sMToVcRNYRJuQwfcUw/Sans-titre.

## Running the code

The front-end folder already contains a Vite project.

1. Open a terminal in `Front-figma`
2. Install dependencies with either:
   - `pnpm install`
   - or `npm install`
3. Start the development server with:
   - `pnpm dev`
   - or `npm run dev`

If you don't have Vite installed globally, it is not required: the project uses the local Vite dependency from `package.json`.
